What is Streamlit, and why use it ?

Streamlit is an open-source Python library that makes it easy to create and share beautiful, custom web apps for machine learning and data science. In just a few minutes you can build and deploy powerful data apps then see it automatically update as you iteratively save the source file.

Installation

Before you get started with Streamlit make sure you have the latest version of python installed in your machine.
Then install streamlit by typing the following code on your terminal 
pip install streamlit
And that's it you're good to go 

Your first web app using Streamlit !

1. Start by creating a new python file in your code editor, and give it a name of area_chart.py

2. Now add the following imports:
               import streamlit as st
               import pandas as pd
               import numpy as np

3. Now lets add a title, to do so add the following line of code:
               st.title('Area chart in Streamlit')

4. Now lets create our chart data
               chart_data = pd.DataFrame(np.random.randn(20, 3), columns=["a", "b", "c"])

Note: 
This line of code is creating a Pandas DataFrame using NumPy's np.random.randn function to generate random values from a standard normal distribution. The resulting DataFrame has 20 rows and 3 columns, named "a," "b," and "c" respectively. Each element in the DataFrame is a random number sampled from a standard normal distribution.

5. And that's it we have got our chart data ready and now to display it we write the following code
               st.area_chart(chart_data)

6. To run the program open a terminal in your code editor and type the following command
              streamlit run area_chart.py
